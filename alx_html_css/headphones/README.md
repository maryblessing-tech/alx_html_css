# Headphones Web Page

## Overview
This project is a responsive web page built using **HTML** and **CSS**, designed to showcase headphones. The layout is based on a Figma design, and it adapts seamlessly to different screen sizes, including mobile devices.

## Features
- Fully responsive design: switches to mobile version when screen width is 480px or less
- Interactive elements:
  - Links change color on hover/active: `#FF6565`
  - Buttons adjust opacity on hover/active: `0.9`
- Centered content with a maximum width of 1000px
- Pixel-perfect design based on Figma mockup
- Smooth typography using **Source Sans Pro** and **Spin Cycle OT** fonts (install if missing)
