# CSS Advanced Project

This folder contains the CSS Advanced task for the ALX Front-End curriculum.

## Description

This project focuses on recreating a Figma design using only HTML and CSS. The goal is to convert an HTML skeleton into a visually accurate, responsive layout by applying advanced CSS techniques.

## Files

- `index.html`  The main HTML file for the advanced page.
- `assets/`     Place images, fonts and any downloaded resources here.

## How to use

1. Duplicate the Figma file to your Drafts to read measurements and typography.
2. Put images from the project into `assets/`.
3. Open `index.html` in your browser to preview.
4. Edit CSS as needed to match the Figma layout and styles.

## Requirements

- Use only HTML and CSS. No frameworks or external libraries.
- Code should pass W3C validation.
- Keep file and folder names lower-case where possible.
- All files should end with a newline.
